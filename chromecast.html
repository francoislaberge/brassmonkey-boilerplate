1. Install the Brass Monkey app. ( iOS / Android )
  2. Setup your Chromecast in developer mode.
  3. Make sure you have the Chromecast Extension.
  4. Cast this tab to your Chromecast.
  5. Open up Brass Monkey with your phone/tablet on the same network.
  6. Select Boilerplate from the list.
  7. Now you should be connected.
  8. Press the button to make mario jump.
